## 缺陷清单

### BUG-001 未登录访问结算页跳转错误（产品缺陷）

| 项目 | 内容 |
|------|------|
| 缺陷ID | BUG-001 |
| 所属模块 | 订单结算 |
| 严重程度 | P1（重要） |
| 优先级 | 高 |
| 发现方式 | 自动化测试（test_checkout_without_login） |
| 发现日期 | 2026-07-03 |

**复现步骤：**
1. 退出登录，确保未登录状态
2. 直接访问 http://localhost/opencart/index.php?route=checkout/checkout

**预期结果：**
页面重定向到登录页 http://localhost/opencart/index.php?route=account/login

**实际结果：**
页面重定向到购物车页 http://localhost/opencart/index.php?route=checkout/cart

**影响范围：**
未登录用户可绕过登录直接访问结算流程

**修复建议：**
在 checkout 路由中增加登录状态校验

### BUG-002 搜索框定位冲突（测试缺陷）

| 项目 | 内容 |
|------|------|
| 缺陷ID | BUG-002 |
| 所属模块 | 测试脚本 |
| 严重程度 | P2（一般） |
| 优先级 | 中 |
| 发现方式 | 自动化测试（test_add_multiple_products） |
| 发现日期 | 2026-07-03 |

**复现步骤：**
1. 登录后加购第一个商品
2. 再次搜索第二个商品

**预期结果：**
搜索框能正常输入第二个关键词

**实际结果：**
报错 strict mode violation: locator("input[name='search']") resolved to 2 elements

**原因分析：**
页面上存在2个搜索框（顶部搜索框 + 搜索结果页搜索框），定位器匹配到2个元素

**修复建议：**
使用更精确的定位器，如 `#input-search` 或 `.form-control-lg`
