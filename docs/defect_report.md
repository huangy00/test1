## BUG-001 空关键词搜索结果提示不友好

| 项目 | 内容 |
|------|------|
| **缺陷标题** | 空关键词搜索显示"无结果"而非提示输入关键词 |
| **所属模块** | 商品搜索 |
| **严重程度** | P2（一般） |
| **优先级** | 中 |
| **发现版本** | OpenCart 4.1.0.3 |
| **测试环境** | Windows + XAMPP + Chrome |
| **发现人** | 测试人员 |
| **发现日期** | 2026-07-02 |

### 复现步骤
1. 打开首页 http://localhost/opencart
2. 搜索框不输入任何内容
3. 按回车搜索

### 预期结果
显示提示信息如"请输入搜索关键词"或显示全部商品

### 实际结果
显示"There is no product that matches the search criteria."（无结果）

### 附件
无

### 分析建议
建议在空关键词时显示提示"请输入搜索关键词"，而非直接搜索
```

## BUG-002 未登录访问结算页未跳转登录页

| 项目 | 内容 |
|------|------|
| **缺陷标题** | 未登录用户访问结算页跳转到购物车页而非登录页 |
| **所属模块** | 订单结算 |
| **严重程度** | P1（重要） |
| **优先级** | 高 |
| **发现版本** | OpenCart 4.1.0.3 |
| **测试环境** | Windows + XAMPP + Chrome |
| **发现人** | 测试人员 |
| **发现日期** | 2026-07-02 |

### 复现步骤
1. 退出登录（确保未登录状态）
2. 直接访问 http://localhost/opencart/index.php?route=checkout/checkout

### 预期结果
页面重定向到登录页 http://localhost/opencart/index.php?route=account/login

### 实际结果
页面重定向到购物车页，显示"Your shopping cart is empty!"

### 附件
无

### 分析建议
结算页面的权限校验可能只检查了购物车是否为空，未检查登录状态。建议在checkout路由中增加登录校验中间件。
```

## BUG-003 需求文档密码长度描述与实际不符

| 项目 | 内容 |
|------|------|
| **缺陷标题** | 需求文档写密码至少4位，实际系统要求6-40位 |
| **所属模块** | 用户注册/登录 |
| **严重程度** | P2（一般） |
| **优先级** | 中 |
| **发现版本** | OpenCart 4.1.0.3 |
| **测试环境** | Windows + XAMPP + Chrome |
| **发现人** | 测试人员 |
| **发现日期** | 2026-07-02 |

### 复现步骤
1. 打开注册页
2. 密码输入"1234"（4位）
3. 填写其他信息
4. 点Continue

### 预期结果
根据需求文档，4位密码应能通过

### 实际结果
提示"Password must be between 6 and 40 characters!"，4位密码被拒绝

### 附件
无

### 分析建议
需求文档与系统实际行为不一致，需产品经理确认：
1. 以系统为准，修改需求文档为6-40位
2. 还是修改系统为4-20位（需求文档写的）
```
