# OpenCart 电商系统 - 需求文档

## 一、项目背景

本系统是一个 B2C 电商平台，支持用户注册登录、商品浏览搜索、购物车管理、下单结算等核心功能。

## 二、功能需求

### 2.1 用户模块

#### 2.1.1 用户注册
- 用户可以通过邮箱注册账号
- 必填字段：名字、姓氏、邮箱、密码
- 邮箱不能重复注册
- 密码至少 4 位
- 注册成功后自动跳转到登录页或账户页
- 需勾选隐私政策才能注册

#### 2.1.2 用户登录
- 使用邮箱 + 密码登录
- 登录成功跳转到账户页面
- 登录失败显示错误提示（密码错误、账号不存在）
- 支持"忘记密码"功能

#### 2.1.3 个人信息
- 登录后可查看/修改个人信息
- 可修改密码
- 可管理收货地址

### 2.2 商品模块

#### 2.2.1 商品列表
- 按分类展示商品（Desktops、Laptops、Phones等）
- 支持分页
- 支持按价格排序

#### 2.2.2 商品搜索
- 支持关键词搜索
- 搜索结果高亮关键词
- 无结果时显示"没有商品"提示
- 搜索框在页面顶部，全站可用

#### 2.2.3 商品详情
- 展示商品名称、价格、图片、描述
- 展示库存状态
- 展示商品评价

### 2.3 购物车模块

#### 2.3.1 加入购物车
- 点击"Add to Cart"按钮加购
- 加购成功显示绿色提示条
- 购物车图标显示商品数量

#### 2.3.2 购物车管理
- 可修改商品数量
- 可删除商品
- 自动计算小计和总价
- 库存不足时提示

#### 2.3.3 购物车入口
- 页面右上角购物车图标
- 点击可查看购物车详情
- 可直接跳转到结算页面

### 2.4 订单模块

#### 2.4.1 结算流程
- 步骤1：填写/选择收货地址（登录用户自动填充）
- 步骤2：选择配送方式
- 步骤3：选择支付方式
- 步骤4：确认订单
- 每步都有"Continue"按钮进入下一步

#### 2.4.2 支付方式
- 支持货到付款（Cash on Delivery）
- 支持免运费结算（Free Checkout）
- 未选择支付方式时，确认按钮禁用

#### 2.4.3 订单确认
- 下单成功显示"Your order has been placed"
- 显示订单号
- 发送订单确认邮件（可选）

#### 2.4.4 订单历史
- 登录后可查看历史订单
- 可查看订单详情
- 可查看订单状态

### 2.5 后台管理（管理员）

#### 2.5.1 商品管理
- 增删改查商品
- 管理商品分类
- 管理库存

#### 2.5.2 订单管理
- 查看所有订单
- 修改订单状态
- 发货管理

#### 2.5.3 用户管理
- 查看注册用户
- 禁用/启用用户

## 三、非功能需求

### 3.1 性能要求
- 页面加载时间 < 3秒
- 搜索响应时间 < 2秒
- 支持 50 并发用户

### 3.2 安全要求
- 密码加密存储
- 防 SQL 注入
- 防 XSS 攻击
- 敏感操作需要登录验证

### 3.3 兼容性要求
- 支持 Chrome、Firefox、Edge 浏览器
- 支持移动端适配

## 四、业务流程图

```
用户访问首页
    │
    ├── 浏览商品 ──→ 商品详情 ──→ 加入购物车
    │                                    │
    ├── 搜索商品 ──→ 搜索结果 ──→ 商品详情 ──┘
    │                                    │
    │                              查看购物车
    │                                    │
    │                              去结算
    │                                    │
    │                         ┌──────────┴──────────┐
    │                         │                     │
    │                    已登录用户              未登录用户
    │                         │                     │
    │                    选择收货地址          跳转登录/注册
    │                         │                     │
    │                    选择支付方式              │
    │                         │                     │
    │                    确认订单                  │
    │                         │                     │
    │                    下单成功                  │
    │                         │                     │
    │                    查看订单                  │
```

## 五、数据字典

| 字段 | 类型 | 说明 |
|------|------|------|
| customer_id | INT | 用户ID，自增主键 |
| firstname | VARCHAR | 名字 |
| lastname | VARCHAR | 姓氏 |
| email | VARCHAR | 邮箱，唯一 |
| password | VARCHAR | 密码，加密存储 |
| telephone | VARCHAR | 电话 |
| product_id | INT | 商品ID，自增主键 |
| name | VARCHAR | 商品名称 |
| price | DECIMAL | 商品价格 |
| stock | INT | 库存数量 |
| order_id | INT | 订单ID，自增主键 |
| order_no | VARCHAR | 订单编号 |
| total_amount | DECIMAL | 订单总金额 |
| status | ENUM | 订单状态：pending/paid/shipped/completed/cancelled |
